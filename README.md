# Voice Input

A native macOS menu bar voice input app planned as a customizable Typeless alternative.

The first version will use:

- A global hotkey to start and stop recording.
- Doubao/Volcengine speech recognition as the first speech-to-text provider.
- DeepSeek as the first text refinement provider.
- A user-editable prompt for personalized text cleanup.
- Clipboard writing and automatic paste into the focused app.

See the design spec in `docs/superpowers/specs/2026-06-11-macos-voice-input-design.md`.
